## How to run

```bash
# Clone this project
$ git clone https://github.com/allexis096/dropbox-clone

# Access this folder via terminal
$ cd dropbox-clone

# Running the app
$ yarn start

# The server init in the port:3000 - access http://localhost:3000
```
